# Local-first AI runtimes

Local-first AI tools run models and agents on the user's own machine instead of a
remote server. They promise lower latency, offline capability, and stronger privacy
because raw context never leaves the device. The trade-offs are setup complexity,
hardware limits, and the work of bridging browser sandboxes to local processes.

Portico exists to make that last part — connecting an app to a locally installed
Agent CLI — boringly reliable.
